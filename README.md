# Veritas: DeepFake Detection & Media Authenticity System

## Overview
Veritas is an AI-powered cybersecurity platform designed to detect deepfake images, videos, and audio in real-time. It combines multimodal AI analysis with real-time desktop monitoring and proactive defense mechanisms to ensure media authenticity

## Key Features
- Image, Video, Audio Deepfake Detection
- Real-time Screen & Browser Content Analysis (Desktop-based)
- Virtual Webcam Detection (Zoom/Meet)
- Adversarial Noise Protection (Aegis Vaccine)

## Architecture
End-to-end multimodal pipeline for detecting deepfake content across image, video, and audio with real-time desktop analysis
